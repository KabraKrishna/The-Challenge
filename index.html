Find the Challenge

The challenge requires you to identify a hidden URL within this HTML Page which yould lead you to the actual challenge.
Follow below rules to identify and build the required URL:
  • The URL is broken into characters and spread across this html page.
  • Your task is to find each character, combine them and get the actual URL.
  • In order to find the URL parts (or characters) find all the div elements that have id = envelop.
  • Every envelop (div) has either of the two children div elements id = letter | id = stamp.
  • Every letter (div) or stamp (div) hold few children span elements with id = character.
  • Every character (span) is associated with data-value attribute, that holds your URL part
  • Find all character (span) within letter (div) of each envelope (div), to build the URL.
  • Example: In the HTML snippet below, x is your URL part.
                            <div id="envelop">
                            <div id="letter">
                                <span id="character"></span> 
                                <span id="character" data-value="x"></span>
                            </div>
                        </div>
  • After concatinating all the parts, you will obtain final URL to reach the actual challenge.
